## 사용법

**[생성]**

-   커맨드 : `npm init -w [경로]`
-   예시 : _`npm init -w packages/react`_ : packges 폴더의 react 프로젝트 생성

**[심볼]**

-   커맨드 : `npm i -s [연결할 프로젝트] -w [연동할 프로젝트]`
-   예시 : _`npm i -s react -w create-react-app`_ : create-react-app 의 react 패키지 연동
